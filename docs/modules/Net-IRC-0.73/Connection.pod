=head1 NAME

Net::IRC::Connection - IRCコネクションのオブジェクト指向インターフェース

=head1 SYNOPSIS

建設場所．このセクションは工事中です．

=head1 DESCRIPTION

このドキュメントはメインのNet::IRCドキュメントの一部分です．もしまだ読
んでいなければ，続ける前に"perldoc Net::IRC"を行ってください．

Net::IRC::Connectionはそのインスタンスが一つのIRCサーバに対する個々のコ
ネクションを表すクラスを定義します．複数のNet::IRC::Connectionオブジェ
クトは一つのNet::IRCオブジェクトで同時に扱うことができます．

=head1 METHOD DESCRIPTIONS

このセクションはまだ作成中ですが，上手くいけばついに次のリリースで書か
れるでしょう．このモジュールの詳細についてはC<irctest>スクリプトとソー
ス自体を参照してください．
