=head1 名前 (NAME)

Number::Phone::JP::Table - 日本の全電話番号の正規表現テーブル

=head1 概要 (SYNOPSIS)

B<このモジュールを直接使用しないこと>

=head1 詳細 (DESCRIPTION)

このモジュールは Number::Phone::JP によって使用する、全ての日本の電話番号
のテーブルを定義しています。
