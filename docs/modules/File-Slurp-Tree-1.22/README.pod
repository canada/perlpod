=head1 名前

File::Slurp::Tree - ネストしたハッシュでファイル・ツリーを丸呑み、吐き出します。

=head1 概要

 # パスaからbにファイル・ツリーを(非効率に)複製します
 use File::Slurp::Tree;
 my $tree = slurp_tree( "path_a" );
 spew_tree( "path_b" => $tree );


=head1 依存性

このモジュールは以下のモジュールに外部依存しています:

 File::Find::Rule
 File::Path
 File::Slurp

=head1 インストール方法

 perl Build.PL
 perl Build test

そして、すべてがうまくいったら

 perl Build install
