=encoding utf-8

=head1 名前

YAML::Base - Base class for YAML classes

YAML::Base - YAML クラス群のベースクラス

=head1 概要

    package YAML::Something;
    use YAML::Base -base;

=head1 説明

YAML::Base is the parent of all YAML classes.

YAML::Base は全ての YAML クラスの親です.
