=head1 名前

 File::BasicFlock - flockでファイルをロックする

=head1 概要

 use File::BasicFlock;

 lock($filename);

 lock($filename, 'shared');

 lock($filename, undef, 'nonblocking');

 lock($filename, 'shared', 'nonblocking');

 unlock($filename);

=head1 説明

flock()を使ってファイルをロックします。ロックされるファイルは
既存のものでなければなりません。これはまさにインターフェースと
いうだけです。
